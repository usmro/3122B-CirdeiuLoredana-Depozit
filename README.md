# Sistem Management Depozit v1.2

Proiect realizat în C++ pentru gestionarea eficientă a stocurilor, monitorizarea tranzacțiilor și analiza fluxurilor de marfă.

## Caracteristici principale:
- **Gestiune stocuri:** Înregistrare intrări (Restock) și ieșiri (Vânzări).
- **Sistem de alerte:** Identificarea automată a produselor aflate sub pragul critic.
- **Istoric tranzacții:** Generare istoric pe baza claselor șablon (Templates).
- **Module inteligente:** Căutare parțială case-insensitive și statistici financiare.
- **Rapoarte:** Export automat în format CSV, TXT și raport managerial în HTML.

## Structura proiectului:
- `/include` - Fișierele header (`.h`)
- `/src` - Implementările metodelor (`.cpp`)
- `/data` - Fișierele de date (`.csv`, `.txt`, `.html`)
